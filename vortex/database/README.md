# Vortex Database Module

Модуль для управления PostgreSQL базой данных с использованием интерактивного интерфейса.

## Основные возможности

1.  **Интерактивный проводник (Explorer)**: Просмотр списка таблиц, их структуры и данных.
2.  **Управление данными**: Просмотр первых 200 строк любой таблицы с пагинацией.
3.  **Управление структурой**: Возможность очистки (`TRUNCATE`) или удаления (`DROP`) таблиц с подтверждением безопасности.
4.  **Статистика**: Общий обзор размера БД и количества объектов.
5.  **SQL Консоль**: Выполнение произвольных запросов с табличным выводом результатов.

## Технические детали

- **Движок**: Использует `vortex.ui.engine.pager` для всех выводов.
- **Драйвер**: `psycopg` (версия 3) с использованием `dict_row` для оптимизации обработки данных.
- **Безопасность**: Все опасные операции требуют ввода `yes` для подтверждения.

## Использование в коде

```python
from vortex.database.db import DatabaseManager

db = DatabaseManager()
db.cmd_tables()  # Запуск интерактивного проводника
db.cmd_stats()   # Показать статистику
```

## Инструкции по разработке

- **Оптимизация**: При получении данных для `pager` всегда используйте `dict_row`.
- **UI**: Не используйте `rich.Table` напрямую, делегируйте это `vortex.ui.engine`.
- **Транзакции**: Всегда проверяйте `conn.info.transaction_status` перед выполнением новых команд.
