# VORTEX UI ENGINE (VUE)

Универсальный интерактивный движок для отображения табличных данных с поддержкой пагинации, навигации клавиатурой и адаптивной верстки.

## Основная функция: `pager()`

```python
from vortex.ui.engine import pager

selected_item = pager(items, title, columns, page_size=5, description=None)
```

### Параметры:
1.  **`items`** (list[dict]): Список словарей с данными. Каждый словарь — это строка таблицы.
2.  **`title`** (str): Заголовок окна (отображается сверху).
3.  **`columns`** (list[dict]): Определение колонок. Каждый словарь может содержать:
    *   `name`: Заголовок колонки (строка).
    *   `key`: Ключ в словаре данных `items`.
    *   `style`: (опционально) Стиль Rich (напр. `"bold green"`).
    *   `no_wrap`: (опционально) Если `True`, текст не будет переноситься.
    *   `min_width`: (опционально) Минимальная ширина терминала, при которой колонка видна.
    *   `render`: (опционально) Функция-коллбэк `func(value, item, width) -> str` для кастомной отрисовки. Принимает текущее значение, весь объект строки и текущую ширину терминала.
4.  **`page_size`** (int): Количество строк на одной странице (по умолчанию 5).
5.  **`description`** (str): (опционально) Текст подсказки под навигацией.

### Особенности:
- **Авто-обработка Boolean**: Значения `True` и `False` заменяются на `✔` и `✘`.
- **Адаптивность**: Колонки с `min_width` автоматически скрываются на узких экранах.
- **Интерактивность**: Навигация стрелками и быстрый выбор по ID.

## Примеры использования:

### Сложный рендеринг с учетом ширины:
```python
def my_renderer(val, item, width):
    if width < 80:
        return f"[bold]{val[:5]}...[/bold]" # Сокращаем, если экран мал
    return f"[bold]{val}[/bold]"

cols = [{"name": "Name", "key": "name", "render": my_renderer}]
```
